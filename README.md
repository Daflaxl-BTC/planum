# Planum – Intelligentes Pflanzen-Tracking

QR-Code scannen, Pflanze fotografieren, KI erkennt sie – und sagt dir genau, wann du gießen, düngen und umtopfen sollst.

## Projekt-Struktur

- `docs/` – Businessplan und Projektdokumentation
- `landing-page/` – React + Vite Landingpage (deployed auf Vercel)
- `claude.md` – Projektkontext für KI-Assistenten

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Vercel (Hosting)

## Entwicklung

```bash
cd landing-page
npm install
npm run dev
```

## Deployment

Push to `main` triggers automatic Vercel deployment.
